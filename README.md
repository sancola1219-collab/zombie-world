# 殭屍世界：黑鴉市事件

第一人稱生存恐怖射擊解謎遊戲——致敬經典生存恐怖系列的原創作品。

**線上遊玩：https://sancola1219-collab.github.io/zombie-world/**

- 三章戰役：月影洋館 → 黑鴉市警察局 → 晨星地下實驗室（開發中）
- 目前版本：階段二「戰鬥示範場」——武器、殭屍與殭屍犬、草藥、打字機存檔
- 純前端 Three.js，程序生成貼圖與合成音效，零外部素材

## 操作

點擊畫面鎖定視角｜WASD 移動｜Shift 奔跑｜左鍵射擊｜R 裝填｜1/2/3 切換武器｜E 互動｜Tab 物品欄｜Esc 暫停

## 開發

```
node --test        # 單元測試
npx http-server .  # 本地執行
```

設計文件與路線圖在 `docs/superpowers/`。
