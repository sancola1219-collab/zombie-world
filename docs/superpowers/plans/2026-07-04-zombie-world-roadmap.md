## 跨階段固定約定（所有計畫共用）

- 純 ES modules；Three.js 僅經 import map 載入 `three@0.170.0`（jsdelivr CDN），除此之外零外部請求
- **邏輯與渲染分離**：遊戲邏輯只跑在 `GameLoop` 固定時步（60Hz）內；rAF 只負責渲染；分頁隱藏時由 `setInterval` 後備驅動邏輯
- **可測性**：`js/game/*`、`js/engine/loop.js`、`js/engine/save.js`、`js/game/world.js` 等邏輯模組**不得 import three、不得碰 DOM**（事件接線集中在 `attach()` 方法與 `main.js`），全部可在 `node --test` 下同步測試
- 測試指令：`node --test tests/`；測試檔命名 `tests/<模組>.test.js`
- 瀏覽器驗證：`.claude/launch.json` 的 `zombie-world` 設定（`npx http-server -p 8123`），用 preview 工具檢查
- 所有 UI 與文案繁體中文；座標約定：x 向東、z 向南、y 向上；yaw=0 面向 −z，`rotation.order='YXZ'`
- 隨機一律用 `js/engine/rng.js` 的 `mulberry32(seed)`（可重現、可測）
- 每完成一個任務就 commit
