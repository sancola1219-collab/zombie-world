# 外部 3D 模型放置處（選用）

把 `.glb` 模型檔放進這個資料夾，遊戲會**自動用它取代**程式生成的模型；沒放就用內建的，不會壞。

| 檔名 | 內容 | 尺寸基準 |
|---|---|---|
| zombie.glb | 殭屍 | 身高約 1.7（單位＝公尺），腳底在原點，面向 -Z |
| dog.glb | 殭屍犬 | 肩高約 0.6，同上 |
| knife.glb / handgun.glb / shotgun.glb / magnum.glb | 手持武器 | 全長 0.25~0.6，槍口朝 -Z |

來源建議：AI 生 3D（Meshy、Tripo 等，匯出 glb）或 CC0 免費模型庫（poly.pizza 篩 CC0、Quaternius、Kenney）。
注意：外部模型不會有內建的四肢擺動動畫（整體移動仍正常）。
